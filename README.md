# -XR-Kinect-
开发时间：2025.5-2026.7
简介：项目通过微软Kinect体感摄像机实现实时动捕，由用户驱动皮影模型的交互
说明：项目基于官方的插件实现了动捕驱动皮影，我们重点解决了用户3d骨骼数据驱动2d皮影模型所导致的z轴翻转扭曲等问题，此外，加了初步的物理交互效果。
 项目整体构建了3层皮影的系统，分别为，动捕层，约束层，物理层。 其中，动捕层通过官方插件实现的最初始的皮影的驱动。约束层通过IKMnager的CCD Solver2D解决z轴扭曲翻转问题，同时，平滑了骨骼联动，整体驱动更加仿真。物理层通过让各个部位通过rigidBody等Unity自带物理系统，每帧去同步约束层来实现基本的驱动，在此基础上，去模拟物理碰撞的效果
 脚本仅实现简单场景切换、关卡选择、以及录屏功能
 分工：程序由我单独开发
